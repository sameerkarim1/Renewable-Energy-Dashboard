# Excel - Renewable Energy Dashboard

This is an interactive Excel dashboard that analyzes global trends in renewable energy. It uses multiple slicers, pivot tables, and advanced charts to explore data on:

* Energy Capacity (MW) by type (Hydropower, Solar, Wind)
* Global Consumption and Investment
* Top 5 Countries for Investment
* Jobs in the Renewable Energy Sector
* Regional Drill-down Analysis
